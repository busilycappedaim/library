# library

A list of all the stuff in my digital hoard. Inside each folder, there will be a text file describing the contents of that folder. I will also include the source from which I obtained those files so you can get them for yourself. Note that some of the files in my hoard might be slighlty altered from the original. For example, I may rename, compress, etc.. the files to better suit my needs. 

> In case the source is no longer available, you can (at some point in the future, once I sort out all the details) request for a specific file and download them directly from me

A majority of this files are anime/manga/manhwa scraped from websites with my self built tool which you can find [here](https://github.com/busilycappedaim/manga-downloader). However, at the current stage, it is not ready for public use.
